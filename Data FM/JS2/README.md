Dit script toont hoe JSON data uit een API kan worden verwerkt in Python. 
De JSON data simuleert een Webex API response. 
Het script zet de JSON om naar een Python dictionary. 
Daarna worden alle items doorlopen. 
Alleen items van het type "person" worden geselecteerd. 
Van deze personen wordt de naam en het e-mailadres getoond. 
Items die geen personen zijn, zoals rooms, worden genegeerd. 
Het doel van deze opdracht is leren werken met API data, JSON parsing en filtering. 
Dit is belangrijk binnen Cloud en DevOps om externe services te integreren.

python3 filter_webex.py
