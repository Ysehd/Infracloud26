README – W2 Webex Automation

In deze oefening werd de Webex REST API gebruikt om via Python een Webex space automatisch aan te maken en daarna opnieuw te verwijderen. Met behulp van de requests-library werd een POST-request verstuurd om een space te creëren en een DELETE-request om deze space te verwijderen. De authenticatie gebeurt via een Bearer access token. Dit experiment toont aan hoe Webex Spaces geautomatiseerd beheerd kunnen worden met REST API’s.


Examenklare uitleg (dit mag je letterlijk gebruiken)

Een Webex Space dient als een samenwerkings- en communicatiekanaal. In deze oefening werd de Webex REST API gebruikt om een space automatisch aan te maken en te verwijderen via Python. Dit toont aan hoe Webex Spaces geautomatiseerd beheerd kunnen worden en geïntegreerd kunnen worden in workflows en automatiseringsprocessen.

Een Webex Space wordt gebruikt als een programmeerbare communicatiecontainer die via API’s automatisch kan worden aangemaakt en beheerd.
